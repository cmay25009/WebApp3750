White text with a border created by using shadows!

This text box is in an overlay container with 6px blur and .2 opacity to make this easier to read
The top border uses a dotted style, while the bottom border is a dash style. The default style is solid, which is why the sides are solid.

This overlaay uses a width of 95% of the margin, but also has a max width of 700px so when on a large screen, it doesn't get absurdly large.
